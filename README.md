# PrefixAutoComplete
Program that outputs all the words that start with a given prefix

The user must supply the size of a list of words, and supply words separated by their given frequency.
The user must also input the number of prefixes followed by the prefixes.
Outputs at most the 10 frequent words that start with a given prefix.

Example input
3
223 hi
123 hon
456 hix
1
hi

Output
hix
hi
